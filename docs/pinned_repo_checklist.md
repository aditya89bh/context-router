# Pinned Repository Checklist

Use this before pinning or sharing the repository publicly.

- [ ] README top section clearly explains the project
- [ ] Status badges are visible near the top of README
- [ ] CLI and example commands run successfully
- [ ] Test suite passes locally
- [ ] Benchmark outputs are current
- [ ] No personal examples remain
- [ ] Key docs are linked or discoverable
- [ ] Repository description is clear
- [ ] Repository topics are set appropriately
- [ ] `docs/releases/v0.1.0.md` exists
